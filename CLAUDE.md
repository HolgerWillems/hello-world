# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is a static GitHub Pages website. There is no build system, package manager, or test suite — changes to `index.html` or `styles.css` are deployed directly via GitHub Pages.

## Files

- `index.html` — the single-page site rendered by GitHub Pages
- `styles.css` — stylesheet linked from `index.html`
- `images/` — static image assets referenced in `index.html`

## Development

Open `index.html` directly in a browser to preview changes locally. No build step or server is required.
